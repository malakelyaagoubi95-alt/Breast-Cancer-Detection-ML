# 🏥 Classification du Diagnostic du Cancer du Sein
**Filière :** ADIA – IISE (Université Ibnou Zohr)  
**Module :** Intelligence Artificielle

---

## 📝 Présentation du Projet
Ce projet utilise des données médicales réelles pour entraîner des modèles de Machine Learning capables de prédire le type de tumeur mammaire. L'objectif est d'aider au diagnostic médical en fournissant une analyse rapide et précise.

## 🛠️ Stack Technique
* **Langage :** Python 🐍
* **Bibliothèques :** Scikit-Learn, Pandas, Seaborn, Matplotlib
* **Modèles entraînés :** Régression Logistique, K-Neighbors, Decision Tree
